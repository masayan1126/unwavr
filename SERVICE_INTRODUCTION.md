# unwavr (アンウェイバー) - 迷わず、今日を積む。

## サービス概要
**unwavr**は、毎日・特定曜日・積み上げ候補（バックログ）のタスクをまとめて管理し、「今日やるべきこと」に集中できるローカルファーストのタスク管理アプリケーションです。
「unwavering（揺るぎない、迷わない）」を語源とし、日々のタスクに迷うことなく取り組める環境を提供します。

## 主な特徴

### 1. 迷わないタスク管理
- **自動分類**: 「毎日やる」「週末だけ」「いつかやる」といったタスクを登録しておけば、その日に必要なタスクだけが自動的に「今日のリスト」に現れます。
- **柔軟なスケジュール**: 曜日指定や期間指定で、週末や連休だけのタスクも漏らしません。

### 2. 積み上げの可視化
- **ポモドーロタイマー**: 作業時間を計測し、見積もりと実績を記録。どれだけ集中できたかが数字で見えます。
- **マイルストーン**: 長期的な目標（例：本を10冊読む、動画を5本投稿する）の進捗を手元で管理できます。

### 3. 最速の導線（ランチャー機能）
- **ワンクリックアクセス**: よく使うWebサイトやアプリ、ローカルフォルダへのショートカットを登録可能。
- **作業開始をスムーズに**: タスクに取り掛かるための「準備」の時間をゼロにします。

### 4. ローカルファースト & プライバシー
- **爆速動作**: データはブラウザ（ローカル）に保存されるため、動作が非常に高速です。
- **安心のプライバシー**: 機密情報はサーバーに送信されません。CSVでのインポート・エクスポートにも対応しており、データの持ち運びも自由です。

## こんな方におすすめ
- ルーティンワークと突発的なタスクが混ざって混乱しがちな方
- 週末や連休にやりたいことをつい忘れてしまう方
- 長期的な目標の進捗を日々実感したい方
- 複数のツールやサイトを行き来して作業するクリエイターやエンジニア

## 始め方
現在、ローカル環境での動作をサポートしています。
1. リポジトリをクローン
2. `npm run dev` で起動
3. `http://localhost:3000` にアクセス
